# Deliverable 4 UML Model Files

These PlantUML files are the original tool-based model sources for Deliverable 4.

Render them with a PlantUML plugin, PlantUML desktop integration, or the PlantUML command line.

Files:

- `01-conceptual-view.puml`
- `02-loan-application-activity.puml`
- `03-loan-application-sequence.puml`
- `04-layered-component-view.puml`
- `05-event-driven-component-view.puml`
- `06-deployment-view.puml`
- `07-bonus-transaction-processing-reference.puml`

Suggested command after installing PlantUML:

```powershell
java -jar plantuml.jar -tpng -o ..\images docs\deliverable4\uml\*.puml
```
